API BASEJUMP: Image Search Abstraction Layer

User stories:

  1. I can get the image URLs, alt text and page urls for a set of images relating to a given search string.
  2. I can paginate through the responses by adding a ?offset=2 parameter to the URL.
  3. I can get a list of the most recently submitted search strings.

The search engine returns image data using the Imgur API

Example - Image Search:

https://paper-wilderness.glitch.me/api/imgsearch/SEARCH_PHRASE

https://paper-wilderness.glitch.me/api/imgsearch/lolcats

Example - Image Search With Page Offset:

https://paper-wilderness.glitch.me/api/imgsearch/SEARCH_PHRASE?offset=OFFSET_NUMBER_BETWEEN_1_AND_165_INCLUSIVE

https://paper-wilderness.glitch.me/api/imgsearch/lolcats?offset=2
